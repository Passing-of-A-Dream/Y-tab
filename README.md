## Y-tab 轻起始页

### 扩展介绍
 - 基于React18、Antd5、valtio、TypeScript构建的一个简易起始页。
 - 启用后，此扩展会将空白页更改为起始页导航

### 兼容性
 - chrome浏览器
 - edge浏览器

### 初始化 & 启动
```pnpm install && pnpm dev```

### 编译

```pnpm build```

out_dir: start-page-dist

### 设置
 - 时间是否显示秒
 - 主题
    - 全局颜色修改
    - 背景图片修改（未开发完成）
 - 打开方式
   - 是否在新标签页打开搜索结果
   - 是否在新标签页打开书签
